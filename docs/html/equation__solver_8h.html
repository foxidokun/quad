enum num_roots solve_quad_eq ( double  a,
double  b,
double  c,
double *  x1,
double *  x2 
)

Solve quadratic equation, write root into given variables (x1&x2) and return number of solutions found or error.

Possible errors

  1. ERANGE_SOLVE – overflow in internal calculations

Overflow checks:

  1. b^2 < DOUBLE_MAX
  2. abs(4*a*c) < DOUBLE_MAX
  3. abs(b^2 - 4*a*c) < DOUBLE_MAX

If there are less than two solutions, unused variables do not change their value. If there is one solution, it is written in x1. If there are two solutions, the order of x1 and x2 is not guaranteed.

Parameters
aQuadratic coefficient
bLinear coefficient
cFree coefficient
x1Pointer to store equation root
x2Pointer to store equation root
Returns
Number of equation roots
