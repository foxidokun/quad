Solve quadratic equation, write roots into given variables (x1\&x2) and return number of solutions found or error. 

Possible errors
\begin{DoxyEnumerate}
\item ERANGE\+\_\+\+SOLVE -- overflow in internal calculations
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Overflow checks\+:
\begin{DoxyEnumerate}
\item b$^\wedge$2 \texorpdfstring{$<$}{<} DOUBLE\+\_\+\+MAX
\item abs(4$\ast$a$\ast$c) \texorpdfstring{$<$}{<} DOUBLE\+\_\+\+MAX
\item abs(b$^\wedge$2 -\/ 4$\ast$a$\ast$c) \texorpdfstring{$<$}{<} DOUBLE\+\_\+\+MAX
\end{DoxyEnumerate}
\end{DoxyNote}
If there are less than two solutions, unused variables do not change their value. If there is one solution, it is written in x1. If there are two solutions, the order of x1 and x2 is not guaranteed.
